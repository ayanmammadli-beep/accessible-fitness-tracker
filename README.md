# Accessible Fitness Tracker

A web-based fitness tracking platform designed for people with health conditions and disabilities.

The system helps users plan workouts and diets while adapting recommendations based on individual health conditions and fitness goals.

## Demo

See the demo video: demo.mov

## Features

### Fitness Tracking
- Track workouts and exercise history
- Filter and recommend exercises based on health conditions
- Goal-based workout planning

### Diet Tracking
- Track nutrition and diet plans
- Filter foods based on dietary needs and health restrictions

### Smart Filtering Logic
- Exercise recommendations adjusted for medical conditions
- Filtering based on fitness goals and user needs

### Real-Time Search
- Search exercises and diet items instantly
- Dynamic filtering for faster navigation

### PDF Export
- Generate downloadable PDF summaries of workout and diet plans

## Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Architecture:** REST API with server-rendered templates

## Motivation

This project was developed for a local client with a focus on **accessible fitness planning** and health-conscious training.

The goal was to create a platform that helps users safely manage workouts and diets while accounting for health conditions and accessibility needs.
